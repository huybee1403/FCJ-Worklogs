On this page, you will follow the learning and working journey week by week throughout the AWS Cloud Journey Bootcamp program.

### 📅 Internship Roadmap:
This program is designed to run over **12 weeks (3 months)**, gradually building knowledge from foundational to advanced levels in AWS cloud infrastructure and software development.

*   **Week 1:** [Tool Setup & Personal AWS Account Configuration](1.1-week1/)
*   **Week 2:** [AWS Global Infrastructure & Budget Setup](1.2-week2/)
*   **Week 3:** [Launching and Managing Amazon EC2 Instances](1.3-week3/)
*   **Week 4:** [Designing Secure Virtual Networks with Amazon VPC](1.4-week4/)
*   **Week 5:** [Amazon S3 Storage & CloudFront Management](1.5-week5/)
*   **Week 6:** [AWS Databases (RDS & DynamoDB)](1.6-week6/)
*   **Week 7:** [AWS IAM Security & Access Control](1.7-week7/)
*   **Week 8:** [Building Serverless Applications with AWS Lambda & API Gateway](1.8-week8/)
*   **Week 9:** [CI/CD Pipelines with AWS CodePipeline](1.9-week9/)
*   **Week 10:** [System Monitoring with CloudWatch & AWS CloudTrail](1.10-week10/)
*   **Week 11:** [Infrastructure as Code with AWS CloudFormation/CDK](1.11-week11/)
*   **Week 12:** [Cost Optimization, Performance Tuning & Bootcamp Summary](1.12-week12/)
